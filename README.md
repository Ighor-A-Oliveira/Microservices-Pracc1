Projeto simples para praticar microservices, o projeto tem dois componentes que se comunicam pelo CloudAMPQ, o componente user reccebe uma request cadastra o usuario no banco, manda uma mensagem assincrone para o CloudAMPQ, o componente email receber essa mensagem assincrona da queue,, cria o email, salve esse email num banco proprio e depois envia esse email para o usuario que foi cadastrado
